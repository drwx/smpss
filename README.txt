安装说明
1.使用数据库导入工具(如：phpmyadmin)导入smss.sql文件
2.修改数据库配置文件config/db.ini.php 文件
3.如果你的服务器不支持伪静态 请在app/base/Constant.class.php中设置REWRITE =false 默认为true
4.在app 下面建立模版缓存目录 v_t 权限必须是可读写
5.app\cache目录必须有可读写权限

确认以上操作你都进行了！
